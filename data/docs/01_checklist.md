# File Completion Checklist (Generic)

**Required**
- Online application form
- Official transcripts (HS or prior colleges)
- Two recommendations (teacher/mentor)
- Personal statement / essay(s)
- Application fee or fee waiver

**Conditional / program-specific**
- Portfolio / audition (arts)
- Additional writing samples (some programs)
- Test scores (only if required or submitted)

**When is a file “complete”?**
A file is complete when all required materials are received and validated (e.g., transcripts match applicant identity, recommenders submitted, etc.).
